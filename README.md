# AI-ML-COURSE
# Foundation of Applied AI & Machine Learning

This repository contains course materials, notes, assignments, and projects developed as part of the Foundation of Applied AI & Machine Learning course at Rajiv Gandhi Institute of Technology (RIT), Kottayam.

The course introduces the fundamental concepts of Artificial Intelligence and Machine Learning, covering mathematical foundations, classical machine learning algorithms, deep learning architectures, generative AI technologies, and deployment of AI applications using modern tools and frameworks.

The repository serves as a structured record of learning, experimentation, and project development throughout the course.
